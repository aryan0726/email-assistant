# AI Email Assistant 🚀

AI-powered email reply assistant that generates smart email responses directly inside Gmail using Gemini AI.

---

## ✨ Features

- AI-generated email replies
- Multiple tone selection
  - Professional
  - Friendly
  - Formal
  - Casual
- Chrome extension integration
- Gmail compose integration
- Spring Boot REST API backend
- Gemini AI integration
- Dynamic prompt engineering
- Real-time response generation

## ⚙️ Setup Instructions

### 1. Clone Repository

```bash
git clone https://github.com/aryan0726/email-assistant.git
```

---

### 2. Backend Setup

```bash
cd backend
```

Add your Gemini API key inside:

```properties
application.properties
```

Run Spring Boot application.

---

### 3. Chrome Extension Setup

- Open Chrome
- Go to `chrome://extensions`
- Enable Developer Mode
- Click `Load unpacked`
- Select `extension` folder

---

### 4. Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
